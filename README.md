# 📝 ToDo List Web App

A simple and responsive **ToDo List Application** built using **HTML, CSS, and JavaScript**.
This project allows users to add, edit, delete, and save tasks using **Local Storage** so tasks remain even after refreshing the page.

---

## 🚀 Features

✅ Add New Tasks
✅ Edit Existing Tasks
✅ Delete Tasks
✅ Save Tasks in Local Storage
✅ Data Persists After Refresh
✅ Responsive & Modern UI
✅ Add Task Using Enter Key

## ⚡ How It Works

### 1. Add Task

* User enters a task in the input field.
* Clicking the **Add Todo** button adds the task to the list.

### 2. Edit Task

* Click the **Edit** button to update an existing task.

### 3. Delete Task

* Click the **Delete** button to remove a task.

### 4. Local Storage

* Tasks are automatically saved in the browser's local storage.
* Tasks remain available even after refreshing the page.
